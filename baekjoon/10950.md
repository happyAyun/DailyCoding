# 10950번 A+B-3

[백준 10950번 A+B-3](https://www.acmicpc.net/problem/10950)

```c++
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;
int main()
{
    int t;
    cin >> t;
    for (int i = 0; i < t; i++)
    {
        int a, b;
        cin >> a >> b;
        cout << a + b << '\n';
    }
    return 0;
}
```
