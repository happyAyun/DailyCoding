# 2741번 N 찍기

[백준 2741번 N 찍기 문제](https://www.acmicpc.net/problem/2741)

```c++
#include <iostream>
#include <algorithm>
#include <cstdio>
#include <string>
#include <fstream>
using namespace std;
int main()
{
    int n;
    scanf("%d", &n);
    for (int i = 1; i <= n; i++)
    {
        printf("%d\n", i);
    }
    return 0;
}
```
